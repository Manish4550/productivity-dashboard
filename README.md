Productivity Dashboard

A lightweight Productivity Dashboard built with HTML, CSS, and JavaScript. Track tasks, mark them complete, and manage your daily productivity easily in the browser.

Live Link - https://productivity-dashboard-iota-six.vercel.app/
Features

Add, complete, and delete tasks

Responsive design for all devices

Dark & Light mode toggle

No backend required

How to Use

Clone the repository:

git clone https://github.com/yourusername/productivity-dashboard.git


Open index.html in your browser.

Start adding and managing tasks.
